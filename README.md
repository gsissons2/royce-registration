# Royce Hotel Registration Card PWA

iPad-based guest registration signing system for The Royce Hotel.

## Features
- Landscape iPad PWA
- Apple Pencil signature support
- Dark/light mode (system preference)
- PDF template parsing for guest data
- Marketing opt-out checkbox
- Single-screen, no-scroll design

## Tech Stack
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- signature_pad
- pdf-parse
- next-pwa

## Development
```bash
npm install
npm run dev
```

## Deployment
Hosted on Vercel.
